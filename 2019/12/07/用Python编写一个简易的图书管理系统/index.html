用Python编写一个简易的图书管理系统

2019-12-07

预期功能

  1.本系统包括登录主界面、管理系统主界面以及各功能分支界面。
  2.登录界面可以实现用户注册、用户登录和退出程序的功能。
  3.管理系统主界面可以实现增加书籍、删除书籍、查阅书籍信息、修改书籍信息、查看所有书籍信息、返回登录界面和退出程序的功能。其中书籍信息包括书名、作者、出版社、定价和数目。
